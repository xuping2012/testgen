## 各模板详细说明

### analyze - 需求分析模板
- **用途**: Phase 1 需求分析阶段，识别功能模块、业务流程、测试点
- **状态**: 使用中
- **代码位置**: `generation_service.py` 第 3209 行
- **功能**: 从需求文档中识别业务功能流程，提取测试点

### review - 模块评审模板
- **用途**: Phase 1 模块评审阶段，评审模块拆分的完整性和合理性
- **状态**: 使用中
- **代码位置**: `generation_service.py` 第 4137 行
- **功能**: 对需求分析的模块拆分和测试点进行评审

### generate - 标准生成模板
- **用途**: Phase 2 用例生成（标准模式）
- **状态**: 使用中
- **代码位置**: `generation_service.py` 第 4861 行
- **功能**: 将测试点转化为详细、可执行的测试用例

### generate_optimized - 分批生成模板
- **用途**: Phase 2 分批生成用例
- **状态**: 使用中
- **代码位置**: `generation_service.py` 第 1205, 4681, 4685 行
- **功能**: 批量生成测试用例，支持分批次处理大量测试点

### generate_with_citation - 带引用生成模板
- **用途**: Phase 2 带引用的用例生成
- **状态**: 使用中
- **代码位置**: `generation_service.py` 第 4677 行
- **功能**: 生成带引用相关用例的测试用例

### rag - RAG检索模板
- **用途**: RAG 检索提示
- **状态**: ❌ 未使用
- **代码位置**: 仅在 `init_db.py` 第 269 行创建，无任何调用
- **功能**: 未实现，可能是为 RAG 页面预留的功能
